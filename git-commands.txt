---Git working flow---
1. git init (folder -> git folder)
2. git add . / fileName (takes git folder to staging area)
3. git commit -m "commit message"


git status (display status)
git diff ()

git log (display commit information details)
git log --oneline (display commit information briefly)

git remote (github repository address)
git remote -v (github repository address full)

git pull (pull everything from github repository to local git folder)

git branch (display all branch and also which branch I am pushing files)
git branch <branchName> (create new branch)

git checkout <branchName> (switch branch)
git checkout -b <branchName> (create and switch to new branch)

git merge <branchName> (merge main branch to another branch)
